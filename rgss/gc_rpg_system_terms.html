RPG::System::Terms

用语的数据类。

超类

  • Object

调用者

  • RPG::System

属性

basic

基本状态，下标如下的字符串数组。

  • 0: 等级
  • 1: 等级(缩写)
  • 2: 体力值
  • 3: 体力值(缩写)
  • 4: 魔力值
  • 5: 魔力值(缩写)
  • 6: 特技值
  • 7: 特技值(缩写)
params

能力，下标如下的字符串数组。

  • 0: 体力上限
  • 1: 魔力上限
  • 2: 物理攻击
  • 3: 物理防御
  • 4: 魔法攻击
  • 5: 魔法防御
  • 6: 敏捷值
  • 7: 幸运值
etypes

装备位置，下标如下的字符串数组。

  • 0: 武器
  • 1: 盾牌
  • 2: 头盔
  • 3: 铠甲
  • 4: 饰品
commands

指令，下标如下的字符串数组。

  • 0: 战斗
  • 1: 撤退
  • 2: 攻击
  • 3: 防御
  • 4: 物品
  • 5: 技能
  • 6: 装备
  • 7: 状态
  • 8: 整队
  • 9: 存档
  • 10: 结束游戏
  • 11: （未使用）
  • 12: 武器
  • 13: 护甲
  • 14: 贵重物品
  • 15: 更换装备
  • 16: 最强装备
  • 17: 全部卸下
  • 18: 开始游戏
  • 19: 继续游戏
  • 20: 退出游戏
  • 21: 回到标题
  • 22: 取消

定义

class RPG::System::Terms
  def initialize
    @basic = Array.new(8) {''}
    @params = Array.new(8) {''}
    @etypes = Array.new(5) {''}
    @commands = Array.new(23) {''}
  end
  attr_accessor :basic
  attr_accessor :params
  attr_accessor :etypes
  attr_accessor :commands
end
