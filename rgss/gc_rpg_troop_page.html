RPG::Troop::Page

战斗事件（页）的数据类。

超类

  • Object

调用者

  • RPG::Troop

属性

condition

条件 (RPG::Troop::Page::Condition）。

span

间隔（0：战斗，1：回合，2：即时）。

list

执行内容（RPG::EventCommand）。

内部类

  • RPG::Troop::Page::Condition

定义

class RPG::Troop::Page
  def initialize
    @condition = RPG::Troop::Page::Condition.new
    @span = 0
    @list = [RPG::EventCommand.new]
  end
  attr_accessor :condition
  attr_accessor :span
  attr_accessor :list
end
