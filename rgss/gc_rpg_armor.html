RPG::Armor

护甲的数据类。

超类

  • RPG::EquipItem

属性

atype_id

护甲类型 ID。

方法

performance

评价护甲的性能。在最强装备指令中使用。

返回物理防御 + 魔法防御 + 所有能力之和。

定义

class RPG::Armor < RPG::EquipItem
  def initialize
    super
    @atype_id = 0
    @etype_id = 1
    @features.push(RPG::BaseItem::Feature.new(22, 1, 0))
  end
  def performance
    params[3] + params[5] + params.inject(0) {|r, v| r += v }
  end
  attr_accessor :atype_id
end
