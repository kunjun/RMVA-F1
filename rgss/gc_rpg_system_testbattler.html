RPG::System::TestBattler

战斗测试中使用的角色数据类。

超类

  • Object

调用者

  • RPG::System

属性

actor_id

角色 ID。

level

等级。

equips

装备，以下面这些为下标的武器 ID 或 装备 ID 的数组。

  • 0: 武器
  • 1: 盾牌
  • 2: 头盔
  • 3: 铠甲
  • 4: 饰品

定义

class RPG::System::TestBattler
  def initialize
    @actor_id = 1
    @level = 1
    @equips = [0,0,0,0,0]
  end
  attr_accessor :actor_id
  attr_accessor :level
  attr_accessor :equips
end
