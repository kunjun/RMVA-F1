RPG::AudioFile

BGM、BGS、ME、SE的超类。

超类

  • Object

属性

name

文件名。

volume

音量（0..100）。BGM 和 ME 的默认值是 100，BGS 和 SE 的默认值是 80。

pitch

音调（50..150）。默认值是 100。

定义

class RPG::AudioFile
  def initialize(name = '', volume = 100, pitch = 100)
    @name = name
    @volume = volume
    @pitch = pitch
  end
  attr_accessor :name
  attr_accessor :volume
  attr_accessor :pitch
end
