RPG::Actor

角色的数据类。

超类

  • RPG::BaseItem

属性

nickname

称号。

class_id

职业 ID。

initial_level

初始等级。

max_level

最终等级。

character_name

行走图的文件名。

character_index

行走图的序号（0..7）。

face_name

脸图的文件名。

face_index

脸图的序号（0..7）。

equips

初始装备。以下面这些为下标的武器 ID 或 装备 ID 的数组。

  • 0: 武器
  • 1: 盾牌
  • 2: 头盔
  • 3: 铠甲
  • 4: 饰品

定义

class RPG::Actor < RPG::BaseItem
  def initialize
    super
    @nickname = ''
    @class_id = 1
    @initial_level = 1
    @max_level = 99
    @character_name = ''
    @character_index = 0
    @face_name = ''
    @face_index = 0
    @equips = [0,0,0,0,0]
  end
  attr_accessor :nickname
  attr_accessor :class_id
  attr_accessor :initial_level
  attr_accessor :max_level
  attr_accessor :character_name
  attr_accessor :character_index
  attr_accessor :face_name
  attr_accessor :face_index
  attr_accessor :equips
end
