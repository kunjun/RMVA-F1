RPG::MapInfo

地图信息的数据类。

超类

  • Object

属性

name

地图名称。

parent_id

父地图编号。

order

编辑器内部使用，地图树状显示的顺序。

expanded

编辑器内部使用，地图树状显示已展开的标记。

scroll_x

编辑器内部使用，横向滚动的位置。

scroll_y

编辑器内部使用，纵向滚动的位置。

定义

class RPG::MapInfo
  def initialize
    @name = ''
    @parent_id = 0
    @order = 0
    @expanded = false
    @scroll_x = 0
    @scroll_y = 0
  end
  attr_accessor :name
  attr_accessor :parent_id
  attr_accessor :order
  attr_accessor :expanded
  attr_accessor :scroll_x
  attr_accessor :scroll_y
end
