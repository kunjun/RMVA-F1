RPG::ME

ME 的数据类。拥有通过 Audio 模块播放自身的功能。

超类

  • RPG::AudioFile

调用者

  • RPG::System
  • RPG::EventCommand

类方法

RPG::ME.stop

停止播放 ME。

RPG::ME.fade(time)

开始淡出 ME。time 以毫秒为单位指定淡出需要的时间。

方法

play

开始播放 ME。

定义

class RPG::ME < RPG::AudioFile
  def play
    if @name.empty?
      Audio.me_stop
    else
      Audio.me_play('Audio/ME/' + @name, @volume, @pitch)
    end
  end
  def self.stop
    Audio.me_stop
  end
  def self.fade(time)
    Audio.me_fade(time)
  end
end
