RPG::Event

超类

  • Object

调用者

  • RPG::Map

属性

id

ID。

name

名称。

x

地图 X 坐标。

y

地图 Y 坐标。

pages

事件页 RPG::Event::Page 的数组。

内部类

  • RPG::Event::Page

定义

class RPG::Event
  def initialize(x, y)
    @id = 0
    @name = ''
    @x = x
    @y = y
    @pages = [RPG::Event::Page.new]
  end
  attr_accessor :id
  attr_accessor :name
  attr_accessor :x
  attr_accessor :y
  attr_accessor :pages
end
