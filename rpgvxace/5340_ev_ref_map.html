地图

启用/禁用显示地图名称

● 功能

控制移动到其他地图时是否显示地图名。

● 设置项目
显示地图名称

要显示则指定[启用]，不要显示则指定[禁用]。


更换地图图块组

● 功能

更改地图的图块组设置。只能更改事件所在的地图。

● 设置项目
图块组

指定更改后的图块组。


更改战斗背景图

● 功能

更改地图的战斗背景设置。只能更改事件所在的地图。

● 设置项目
文件列表

指定更改后背景（左侧）和地板（右侧）的图像。当前指定的图像可以在右侧的预览区域确认。

● 备注

・ 在战斗事件中执行时，背景会从下次战斗开始更改。


更改地图远景图

● 功能

更改地图远景图的设置。只能更改事件所在的地图。

● 设置项目
图像

指定更改后的图像。

横向循环

启用后，图像会横向循环显示。要自动卷动的话在[自动卷动]中指定速度（-32~32，0为停止）。

纵向循环

启用后，图像会纵向循环显示。要自动卷动的话在[自动卷动]中指定速度（-32~32，0为停止）。


获取指定位置的信息

● 功能

获取一个与地图特定位置相关的值并代入变量。可以获取的只有事件所在的地图的信息。

● 设置项目
接受信息的变量

指定将获得的值代入的变量。

信息种类

指定要获取的信息种类。

位置

指定获取信息的位置。要指定特定位置时选择[直接指定]，点击[…]会弹出窗口，在其中指定移动的目的地。要用地图ID和坐标指定的话选择[变量指定]，指定[X 坐标][Y 坐标]要使用的变量。
